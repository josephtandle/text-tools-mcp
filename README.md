# Text Tools MCP Server

A comprehensive text manipulation toolkit for AI workflows, exposed as an MCP server.

## Tools

| Tool | Description |
|------|-------------|
| `slugify` | Convert text to URL-safe slugs |
| `word_count` | Count words, characters, sentences, and estimate reading time |
| `case_convert` | Convert between camelCase, snake_case, kebab-case, UPPER, lower, Title Case |
| `base64_encode` | Encode text to Base64 |
| `base64_decode` | Decode Base64 to text |
| `url_encode` | URL-encode a string |
| `url_decode` | URL-decode a string |
| `markdown_to_html` | Convert basic Markdown to HTML |
| `truncate` | Truncate text to N characters with ellipsis |
| `reverse` | Reverse a string |
| `extract_emails` | Extract email addresses from text |
| `extract_urls` | Extract URLs from text |
| `count_occurrences` | Count occurrences of a substring |
| `pad_string` | Pad a string to a target length |
| `repeat` | Repeat a string N times |
| `strip_html` | Remove HTML tags from text |
| `lorem_ipsum` | Generate placeholder text |

## Usage

```bash
npm install
npm start
```
